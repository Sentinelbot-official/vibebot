üîß Common Issues & Solutions

‚ùå Bot not responding to commands

Solutions:

  • Check if the bot is online (green status in member list)
  • Verify you're using the correct prefix (default is //)
  • Make sure the bot has permission to read and send messages in the channel
  • Try using the command in a different channel
  • Use //help to verify the bot is working

üö´ "Missing Permissions" error

Solutions:

  • Ensure the bot has Administrator permissions (recommended)
  • Check that the bot's role is above the roles it needs to manage
  • Verify channel-specific permissions for the bot
  • Re-invite the bot with proper permissions using the invite link

üí∞ Economy commands not working

Solutions:

  • Make sure you've used //daily at least once to initialize your account
  • Check if you're on cooldown with the command
  • Verify you have enough balance for the action
  • Try //balance to check your account status

üéÆ Games or commands timing out

Solutions:

  • This usually means the bot is under heavy load
  • Wait a few moments and try again
  • Check the bot's status on our status page
  • Report persistent issues on GitHub

üîë Premium activation not working

Solutions:

  • Make sure you have "Manage Server" permission
  • Verify you're using the correct activation key format
  • Check that the key hasn't already been used
  • Contact us on Ko-fi if the issue persists
  • Use //premium to check your server's current status

‚ö†Ô∏è Moderation commands not working

Solutions:

  • Ensure you have the required permissions (Ban Members, Kick Members, etc.)
  • Check that the bot's role is higher than the target user's role
  • Verify the bot has the necessary server permissions
  • You cannot moderate users with Administrator permission

üìä Stats or leaderboards not updating

Solutions:

  • Stats update every few minutes, not instantly
  • Try again after a few minutes
  • Use //rank to see your current XP and level
  • Report persistent issues on GitHub

ü§ñ AI commands not working

Solutions:

  • AI commands require Premium or VIP subscription
  • Check your server's premium status with //premium
  • Verify your prompt doesn't violate content policies
  • AI services may occasionally be unavailable - try again later
