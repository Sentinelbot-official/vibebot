# Privacy Policy

**Last Updated:** December 31, 2025

This Privacy Policy explains how we collect, use, store, and protect your data when you use our Discord bot ("the Bot").

## 1. Information We Collect

### 1.1 Automatically Collected Data

When you use the Bot, we automatically collect:

**User Data:**

- Discord User ID (unique identifier)
- Username and discriminator (for display purposes only)
- Server/Guild membership information

**Server Data:**

- Discord Server/Guild ID
- Server name (for logging purposes)
- Channel IDs (for command execution and logging)

**Usage Data:**

- Commands executed
- Timestamps of interactions
- Error logs and debugging information

### 1.2 User-Generated Data

Data you actively provide through Bot commands:

**Economy System:**

- Virtual currency balance (wallet and bank)
- Daily reward claim timestamps and streak data
- Work, fish, hunt, dig command usage timestamps
- Transaction history (gifts, purchases, robberies)
- Shop inventory and purchases
- Total earned and spent statistics
- Crime and gambling activity

**Leveling System:**

- Experience points (XP)
- Level information
- Message count
- Last XP gain timestamp
- Level role reward configurations

**Moderation System:**

- Warning records (moderator ID, reason, timestamp, case ID, count)
- Moderation actions (kicks, bans, timeouts, mutes, purges)
- Auto-moderation violations and actions taken
- Moderation case history
- User infraction counts

**Server Settings:**

- Auto-moderation configuration (spam, links, invites, caps, mentions)
- Anti-raid protection settings
- Level role rewards
- Reaction role configurations
- Custom prefix per server
- Welcome/goodbye message settings
- Suggestion channel configuration
- Starboard settings
- Ticket system configuration
- Verification system settings
- Server log channel

**AFK System:**

- AFK status and reason
- AFK timestamp

**Giveaway System:**

- Active giveaway data (prize, duration, winners, entries)
- Giveaway history

**Custom Tags:**

- Tag names and content
- Tag creator and usage count

### 1.3 Message Content

- **Temporary Processing:** Message content is processed temporarily for:
  - Auto-moderation features (spam detection, link filtering, caps detection, mass mentions)
  - Command parsing and execution
  - Leveling system (XP gain)
  - Snipe commands (last 60 seconds only)
- **Limited Storage:**
  - Deleted/edited messages cached for 60 seconds for snipe commands
  - Message content for active trivia games (until answered or timeout)
- **Not Permanently Stored:** Message content is NOT stored in our database long-term
- **Immediate Deletion:** Message content is discarded after processing or cache expiry

## 2. How We Use Your Data

We use collected data for:

**Bot Functionality:**

- Providing economy, leveling, and moderation features
- Tracking user progress and statistics
- Enforcing server rules through auto-moderation
- Assigning roles based on levels or reactions

**Service Improvement:**

- Debugging and fixing errors
- Analyzing usage patterns to improve features
- Monitoring Bot performance and uptime

**Compliance:**

- Complying with Discord's Terms of Service
- Responding to legal requests (if required)

## 3. Data Storage and Security

### 3.1 Storage Location

- Data is stored in a SQLite database on our secure servers
- Backups are created automatically every 6 hours
- Last 7 backups are retained, older backups are deleted

### 3.2 Security Measures

- Database files are stored with restricted access permissions
- Server access is limited to authorized administrators only
- Regular security updates and patches are applied
- Encrypted connections for all Bot communications

### 3.3 Data Retention

- **Active Users:** Data is retained as long as the Bot is in use
- **Inactive Users:** Data may be retained indefinitely for historical purposes
- **Deleted Servers:** Server data is retained for 30 days after Bot removal, then deleted
- **Backups:** Backup files are retained for 7 days, then automatically deleted

## 4. Data Sharing

- Share data with other Discord bots

We MAY share data:

- With Discord (as required by their API and Terms of Service)
